What Projects have I worked on?

uOttawa Roboboat

From 2017 to 2019, I served as a software co-lead, followed by community manager for the University of Ottawa Robotics team Roboboat designing a autonomous surface vehicle (i.e boat) that could compete in a variety of navigation tasks based on image recognition and course navigation. As co-lead for software I designed the architecture for the system and worked to integrate parts of the system using the ROS enviroment and Python and C++. I further aided in implementing XBEE radio modules to communicate between the system computer and ground station using Arduino interfaces and IEEE 802.15.4 communication and the algorithms for gate and cluster navigation of the robot using a modified A* algorithm and a designed deterministic algorithm respectively.

Capstone: Hygeing Healthcare Montior

For my capstone project I worked in a team of 5 designing a health managment monitor using Raspberry Pi and a cloud server backend that could monitor the hygeine habits of a individual using a none intrusive sensor array. This project was inspired by consultations with engineers from a local medical centre and were based on needs analysis from the "Emerging Technologies to Support an Aging Population" report by the US National Science & Technology Council with the origional end goal of testing implementation in a model retirement home lab at a local facility. Due to the COVID-19 outbreak, the complete system was changed to focus on home hygeine and implementation techniques were made to allow further sensors for tracking washing hands and other general hygeine requirements. My work in this project included designing the architecture for the local device, development of the REST API and the communication pipeline of data collection, interpretation and distribution to the backend system, as well as a optional sensor for a computer vision to detect when someone was entering a room.

Extra Curricular

COVID-19 Charity Patch

As the pandemic began and summer work was unsure, I spent some time looking at my creative side. A student culture that many are unaware is the "Engineering Coveralls" and patch culture that is in many canadien engineering schools. As someone who has used this culture of trading patches as a way to network for the past 4 years, decided I wanted to make a patch that no only was a fun trading item but also did some good.

Partnering with the Guelph and uOttawa Engineering Societies, I created the "Don't Touch my PPE" nurses charity patch, based on a old Guelph patch oriented around engineering safety and raised a total of $ 416.75 alongside the nearly $500+ donations from Guelph and uOttawa that was given to local food banks, red cross and mental health organizations.

Movember 2020 drive

From the sucess of the covid charity patch, I partnered with the uOttawa Engineering Student Society to create a Movember 2020 campain with a charity patch based on the uOttawa EngSoc mascot "Otis the bunny" as well as collected patches from across Canada for a charity auction. Overall this event raised over $1025 for the Movember Foundation.
